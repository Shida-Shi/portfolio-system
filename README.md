# 🗂 ポートフォリオ集計システム

> **「Excelで管理していた売上データを、Pythonで自動化・効率化したシステムです」**

---

## 📌 これは何？

店舗ごとの月別売上データを管理・集計・出力するシステムです。

もともとExcel VBAで作っていた業務ツールを、**Pythonでリファクタリング（作り直し）**しました。

---

## 🤔 なぜPythonに切り替えたの？

| | Excel VBA | Python（このシステム） |
|---|---|---|
| 動く場所 | Excelの中だけ | どのPCでも動く |
| データ量 | 多いと重くなる | 何万件でも高速 |
| 拡張性 | Excelに依存 | 他システムとも連携できる |
| 保守性 | Excelが壊れると終わり | コードとデータが分離 |

---

## 💻 使用技術

| 技術 | 用途 |
|---|---|
| Python 3.12 | メインプログラム |
| pandas | データの読み書き・集計 |
| openpyxl | Excelファイルの生成・グラフ作成 |
| pathlib | ファイルパスの管理 |

## ⚙️ 機能一覧

### 1. データ登録（1件）
店番を入力するだけで、マスタから**会社名・店舗名・グループ名を自動補完**して登録します。

```
店番を入力してください: 0001
-> 判定: 株式会社A / 東日本支部_1号店
商品名 (空欄なら '商品1'):
税抜金額: 50000
```

### 2. テンプレート生成
一括登録用のExcelテンプレートを自動生成します。

### 3. 一括登録（Excelから）
テンプレートに入力した**何件でも一括登録**できます。マスタにない店番は自動でスキップしてエラー一覧を表示します。

```
【一括登録完了】
✅ 登録成功: 48件
⚠ スキップ（マスタ未登録）: 2件
  店番: 9998, 9999
```

### 4. 集計表示
会社名・商品名・按分率を指定して月別集計を表示します。

### 5. 一括出力（グラフ付き）
全会社・全商品の組み合わせを**月別集計表＋棒グラフ付きExcel**で一括出力します。

### 6. バックアップ
蓄積データを日時付きファイル名で自動保存します。

### 7. 年度更新
バックアップを自動作成してからデータをリセットします。

## 🚀 使い方

### 初回セットアップ
1. Pythonをインストール（3.12推奨）
2. 必要なライブラリをインストール
```bash
pip install pandas openpyxl
```
3. `portfolio - コピー.xlsx` のマスタシートに店舗情報を入力

### 日常の使い方
`ポートフォリオ集計システム起動.bat` をダブルクリックするだけ！

---

## 🔄 システムの流れ

```
マスタ（店舗情報）
        ↓ 照合・自動補完
  データ登録 or 一括登録
        ↓
  蓄積データ.xlsx（データが積み上がっていく）
        ↓
  一括出力（グラフ付きExcelを自動生成）
        ↓
  報告用ファイルを取引先に提出
```

---

## 📊 出力サンプル

一括出力で生成されるExcelには以下が含まれます：

- 月別の税抜金額・按分金額の一覧表
- 月別売上の棒グラフ（自動生成）
